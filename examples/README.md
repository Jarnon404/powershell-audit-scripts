This folder contains synthetic examples only.

The sample files are intended to demonstrate the type of output this repository may generate without exposing real customer, tenant, device, user or infrastructure data.

Do not place real generated reports in this folder unless they have been fully reviewed and sanitized for public sharing.

Generated reports may contain sensitive operational information such as device names, user names, group names, tenant identifiers, server names, IP addresses or security findings.
