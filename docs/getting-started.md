# Getting started

## Requirements

Typical requirements depend on the script being used.

Common examples:

- PowerShell 5.1 or PowerShell 7+
- Microsoft Graph PowerShell modules
- Exchange Online PowerShell module
- Active Directory PowerShell module
- Read-only permissions suitable for the target environment

## Recommended use

1. Review the script before running it.
2. Test in a lab or non-production environment.
3. Run with least-privilege permissions.
4. Review generated reports before sharing.
5. Do not commit generated output files to the repository.

## Example workflow

```powershell
cd C:\Scripts
.\Invoke-ExampleAuditReport.ps1
```

Replace the example script name with the specific script you intend to run.
